## Features
- **Responsive Design**: Works seamlessly on desktop and mobile.
- **Futuristic UI**: Neon glow, animated gradient background, and floating particles.
- **Auto Color Pulse**: Dynamic accent color cycling enabled by default.
- **Countdown Timer**: Displays time remaining until launch.
- **Notify-Me Form**: Collects email addresses for launch notifications.
  - **Mailto fallback** (static)
  - **Optional API endpoint** for backend integration
- **Accessibility & SEO**: ARIA roles, semantic HTML, and `noindex` meta tag.
